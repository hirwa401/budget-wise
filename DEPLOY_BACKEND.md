# Deploy to Heroku
# 1. Install Heroku CLI: https://devcenter.heroku.com/articles/heroku-cli
# 2. Login: heroku login
# 3. Create app: heroku create your-budgetwise-backend
# 4. Add MySQL database: heroku addons:create jawsdb:kitefin
# 5. Deploy: git push heroku main
# 6. Get database URL: heroku config:get JAWSDB_URL
# 7. Set environment variables based on the database URL
# 8. Update API_BASE in app.js to your Heroku app URL
